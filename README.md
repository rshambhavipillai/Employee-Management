# Employee Management System 👨‍💼

This repository hosts a robust **Employee Management System** developed in Java, designed for streamlined deployment via Docker containers and integrated with **phpMyAdmin** for intuitive database management.[page:48] It provides full CRUD operations for employee records, including personal details, contact information, job roles, and employment history.[page:48]

---

## Features

- **Java backend**: High-performance, scalable application built with Java.[page:48]  
- **Dockerized deployment**: Application and database are containerized for consistent and portable environments.[page:48]  
- **phpMyAdmin integration**: Web-based UI for managing the MySQL/MariaDB database.[page:48]  
- **Full CRUD operations**: Create, read, update, and delete employee data through RESTful endpoints.[page:48]  
- **Comprehensive data model**: Stores names, email, phone, hire date, job title, and salary details.[page:48]

---

## Technologies Used

- **Java** – Core backend language.[page:48]  
- **MySQL / MariaDB** – Relational database for employee data storage.[page:48]  
- **phpMyAdmin** – Web interface for database administration.[page:48]  
- **Docker & Docker Compose** – Containerization and orchestration of app, DB, and phpMyAdmin.[page:48]
